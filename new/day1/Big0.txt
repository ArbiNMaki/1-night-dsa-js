*Objects - BigO*
- Insert O(1)
- Remove O(1)
- Access O(1)
- Search O(n)
- Object.keys() O(n)
- Object.values() O(n)
- Object.entries() O(n)

*Array - BigO*
- Insert / Remove at end O(1)
- Insert / Remove at beginning O(n)
- Access O(1)
- Search O(n)
- Push / Pop O(1)
- Shift / Unshift / Concat / Slice / Splice O(n)
- forEach / Map / Filter / Reduce O(n)
